# MercaCredit — Frontend

Aplicación web desarrollada en Angular 21 para el sistema de gestión financiera de cupos

## Tecnologías
- Angular 21
- TypeScript
- Html
- CSS

## Requisitos
- Node.js 24+
- Angular CLI 21

## Estructura
Organizada bajo la metodología Atomic Design: átomos, moléculas, organismos, templates y páginas.
